之前用Deepin的时候水了一大批垃圾博客，自己在csdn写的博客质量差，内容重复且很误导人。今天把博客链接到github，csdn的就全删了不影响百度搜索了。

搞完之后就弃坑了，以后也许只会在虚拟机里看看UOS做成什么样了。
